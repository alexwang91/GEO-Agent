# Public Technical Preview

GEO-Agent is an AI visibility workbench for fixture-safe audits, manual-import evidence, source classification, diagnosis, optimization tasks, retest planning, and report v2 output.

## Scope

- Project setup and query review content contracts.
- Manual-import and fixture-safe evidence workflows.
- Citation parsing, source classification, citation absorption, and claim fidelity checks.
- Diagnosis taxonomy, task plan exports, retest planning, and report v2 checks.
- Industry templates for B2B SaaS, Ecommerce, and Local Service scenarios.

## Guardrails

- No live provider credentials are required.
- CI does not run live provider, crawler, browser, or network-dependent paths.
- Planned providers remain planned.
- OpenAI-compatible output is not ChatGPT Search.
- Low-sample conclusions are directional until repeated sampling clears the noise floor.

## Quick start

1. Define a project profile.
2. Review query clusters and priorities.
3. Add fixture or manual-import evidence.
4. Run the fixture-safe audit path.
5. Review dashboard, drilldown, diagnoses, task plan, and report v2 output.
